# Recipe: Build an Email Sequence

⏱️ **Time:** 4 minutes  
📊 **Difficulty:** Beginner  
🔌 **Plugin:** email-marketing

---

## What You'll Get
- 5-7 email sequence
- Subject lines for each email
- Personalization tokens
- Timing recommendations
- A/B test variations

---

## Quick Steps

1. **Install plugin** (one-time):
   ```
   /plugin install email-marketing
   ```

2. **Run command**:
   ```
   /email-marketing:design-campaign
   ```

3. **Describe your sequence**:
   ```
   "Cold outreach sequence for SaaS prospects, 5 emails over 2 weeks"
   ```

4. **Load into your email tool**

---

## Example Sequences

**Cold Outreach:**
```
Create a 5-email cold outreach sequence for B2B SaaS prospects.
Target: VP of Sales at Series A companies
Goal: Book a demo
Tone: Professional but friendly
```

**Product Launch:**
```
7-email product launch sequence for existing customers.
Announce new feature, share benefits, drive adoption
```

**Nurture Campaign:**
```
10-email nurture sequence for trial users.
Educate on features, share case studies, convert to paid
```

**Re-engagement:**
```
3-email re-engagement sequence for inactive users.
Win them back with new features and special offer
```

---

## Pro Tips

✅ Specify number of emails and timeframe  
✅ Define clear goal (demo, trial, purchase)  
✅ Include target persona details  
✅ Request A/B test variations

---

## Email Sequence Template

**Email 1 (Day 0):** Introduction + value prop  
**Email 2 (Day 3):** Social proof + case study  
**Email 3 (Day 7):** Address objections  
**Email 4 (Day 10):** Urgency + special offer  
**Email 5 (Day 14):** Final attempt + breakup

---

## Advanced Options

**With personalization:**
```
Include personalization for: company name, industry, pain points,
recent news (funding, hiring, product launches)
```

**With A/B testing:**
```
Create 3 subject line variations for each email for A/B testing
```
