# Recipe: Create a Blog Post

⏱️ **Time:** 3 minutes  
📊 **Difficulty:** Beginner  
🔌 **Plugin:** content-marketing

---

## What You'll Get
- Complete 1500-2000 word blog post
- SEO-optimized with keywords
- Proper heading structure
- Meta description
- Internal linking suggestions

---

## Quick Steps

1. **Install plugin** (one-time):
   ```
   /plugin install content-marketing
   ```

2. **Run command**:
   ```
   /content-marketing:generate-blog
   ```

3. **Enter your topic**:
   ```
   "How to automate sales prospecting with AI"
   ```

4. **Review and publish**

---

"5 Ways AI is Transforming B2B Sales Teams"

---

## Pro Tips

✅ Include target keywords in your topic  
✅ Specify word count if needed (1500-2000 default)  
✅ Mention target audience for better personalization  
✅ Request specific examples or case studies

---

## Advanced Options

**With SEO focus:**
```
/content-marketing:generate-blog "sales automation guide" 
--keywords "sales automation, AI prospecting, lead generation"
--word-count 2500
```

**With specific angle:**
```
"Write a blog post about sales automation from the perspective 
of a VP of Sales at a Series B SaaS company"
```
